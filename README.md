# This is educatation project `grep`

### How to install:
#### core `grep`
```
cargo install core-grep
```
#### grep `search`
```
cargo add grep-search
```
### Example of using `grep`:
```
grep [options] [query] [path]
```
#### Options:
                  -i -- ignore case
                  -c -- count lines
                  -help -- help
                  -version -- show version grep

#### Version `grep` -- 0.1.6
